# UiLoadingPage

UiLoadingPage shows a full-screen page with a spinner. There are no props to control visibility, it is expected to replace the rendering of the page based on criteria.

## Example

<Playground>
  <IFramePlayground minHeight={400} maxHeight={800}>
    <UiLoadingPage isOpen={true} />
  </IFramePlayground>
</Playground>

### Example Usage

```
const App = () => {
  const [isReady, setIsReady] = useState(false)
  return isReady ? <RealComponent /> : <UiLoadingPage />
}
```

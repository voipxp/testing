# UiCard

Renders a uniform Card with a title, optional buttons, and the content.

## Example

<Playground>
  <UiCard
    title="Card Title"
    buttons={
      <>
        <UiButton color="info" icon="add" size="small" />
        <UiButton color="light" icon="cancel" size="small" />
      </>
    }
  >
    <p>This is a child paragraph</p>
  </UiCard>
</Playground>

# UiClose

A simple close button that wraps the child component on the top-left corner.

## Example

<Playground>
  <UiClose onClick={() => console.log('closed card')} />
  <UiCard title="Example Card">

  </UiCard>
  <hr />
  <UiClose onClick={() => console.log('closed box')} />
  <ExampleBox text="Example Box">

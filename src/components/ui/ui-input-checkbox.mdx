# UiInputCheckbox

Displays a form checkbox, styled according to Odin styles.

## Example

<Playground>
  <UiInputCheckbox
    name="check1"
    label="Not Checked"
    checked={false}
    onChange={() => console.log('changed')}
  />
  <UiInputCheckbox
    name="check1"
    label="Checked"
    checked={true}
    onChange={() => console.log('changed')}
  />
  <UiInputCheckbox
    disabled
    name="check1"
    label="Disabled Checked"
    checked={true}
    onChange={() => console.log('changed')}
  />
</Playground>

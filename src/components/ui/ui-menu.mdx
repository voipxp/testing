# UiMenu

Renders a bulma menu based on routes. You must pass in the **menu** prop which contains an array of sections and menu items. The Menu sections and items within each section will be in the same order that is passed in via the menu prop.

UiMenu relies on react-router for navigation and must be within a Router context.

## Example

<Playground>
  <HashRouter hashType="hashbang">
    <Route
      path="/"
      render={props => (
        <UiMenu
          {...props}
          menu={[
            {
              section: 'First',
              items: [
                { name: 'Link 1', path: 'link-1', component: ExampleBox },
                { name: 'Link 2', path: 'link-2', component: ExampleBox }
              ]
            },
            {
              section: 'Second',
              items: [{ name: 'Link 3', path: 'link-3', component: ExampleBox }]
            }
          ]}
        />
      )}
    />
  </HashRouter>
</Playground>
